# officeOnline
### 多人在线文档编辑

##### 插件
* maven
* ssm
* websocket(创建在线聊天室)
* {[sheetJS](https://github.com/SheetJS/sheetjs)}(读取excel文件)
* {[MyExcel](https://gitee.com/beany/myExcel)}(显示表格数据)

##### 功能

* 上传文档
* 多人在线编辑文档
* 多人编辑间的交互
* 多sheet
* 导出文档
